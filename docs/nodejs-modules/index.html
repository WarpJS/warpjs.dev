Using Node.js Modules

From the samples/hello-world example of the Quick Start, we will create a concrete example using a Node.js module and deploy it on serverless infrastructure.

In the following example, we will create a simple HTTP proxy to request a public REST API with the popular axios module.

Import Module

In your terminal, add the axios module to your package.json:

npm install axios

Call module

Add an async function to fetch a user list from an API and call it on page load:

src/index.js

// ...

// warp function
const fetchUsers = async () => {
  'warp +server -client'

  const axios = require('axios')

  // fetch users from API
  const { data } = await axios.get('https://jsonplaceholder.typicode.com/users')

  return data
}

// on load
warper.call(fetchUsers).then(response => {
  result.innerHTML += `
    <h2>User List</h2>
    <div>${response.map(user => `${user.id} - ${user.name}`).join('<br>')}</div>
  `
})

Run and Deploy

Run your project on dev:

npm start

And deploy it on serverless:

npm run build
npm run deploy
