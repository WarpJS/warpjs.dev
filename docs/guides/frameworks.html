Framework integration with WarpJS

This guide shows you how to integrate WarpJS inside the most popular frameworks like Angular, React, Vue.js. As you know, each framework offers a command line to help you to start.

Initialize project

First, create a main project:

Angular
React
Vue.js
# https://angular.io/cli

$ npm install -g @angular/cli
$ ng new <project-name>
# https://create-react-app.dev/docs/getting-started/

$ npx create-react-app <project-name>
# https://cli.vuejs.org/

$ npm install -g @vue/cli @vue/cli-service-global
$ vue create <project-name>

Setup WarpJS server

We will create a Basic HTTP Proxy in Node.js in order to:

  • Fetch data from a REST API (https://jsonplaceholder.typicode.com/users)
  • Filter data to avoid to request too much data as client side
  • Add random avatar in response

Create WarpJS server project

Create a server directory at the root of your main project and initialize a new Node.js project inside it:

$ cd <project-name>
$ mkdir server
$ cd server
$ npm init

And install the Axios library we need:

$ npm install axios

Create serverless function

Create index.js as follow:

const axios = require("axios");

const getUsers = async () => {
  // fetch users from API
  const { data } = await axios.get("https://jsonplaceholder.typicode.com/users");

  // Pick attributes and add photo
  return data.map(({ id, name, email, phone, company, address }) => ({
    id,
    name,
    email,
    phone,
    city: address.city,
    work: company.name,
    photo: "https://randomuser.me/api/portraits/lego/" + (id % 10) + ".jpg",
  }));
};

// getUsers will be the function that you will can call directly in the framework
module.exports = { getUsers };

Configure WarpJS

Create a warp.config.js file:

module.exports = {
  project: "project-server-name",
  output: {
    format: "node-module",
    // path to the "node_modules" directory of your main project
    projectPath: "../",
    // module name to import it in your main project
    name: "warp-server",
  },
};

Thanks to the node-module output format, WarpJS will generate an npm package named warp-server.

Why this option?

  • node_modules is a safe directory: each framework uses tools like Babel, ESLint, ... and here, you are sure that the generated code will not be transpiled or something else.
  • Really easy to import: no relative path to know, you can directly do: import WarpServer from "warp-server"

Setup WarpJS in project

You have to integrate the WarpJS CLI in the main project for:

  • Start the WarpJS emulator in development mode to simulate serverless function on localhost
  • Build WarpJS client library in order to call the serverless function
  • Deploy the serverless function and optionally the static build of the main project

Install dependencies:

$ npm install @warpjs/engine
$ npm install warp npm-run-all --save-dev

Call serverless function in your code

First, you have to import the WarpJS engine once. We recommend to initialize it in the entry point of your main project:

Angular
React
Vue.js
// src/main.ts

import "@warpjs/engine";
// src/index.js

import "@warpjs/engine";
// src/main.js

import "@warpjs/engine";

Then, import the npm module to init it and call the serverless function:

import WarpServer from "warp-server";

const { getUsers } = new WarpServer();

const users = await getUsers();

For a complete integration, see "Code Samples" section.

Run

Login to WarpJS

Run the warp login command to allow authenticating to your WarpJS account on next steps:

$ npx warp login

Create a project in your WarpJS Console

You have to create a project in your WarpJS console. Sign In and add a project with the same name as your warp.config.js.project attribute.
