Commande-line Interface (CLI)

Development

To start in development mode on your local environment:

npm start

The development mode has a live reload capability on every code change.

Production

To deploy your WarpJS functions in production to our preselected serverless infrastructures, configure your build (see the api/webpack section) and use the following command:

npm run deploy
