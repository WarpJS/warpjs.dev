Requirements

WarpJS SDK has the following requirements:

  • Install Node.js latest 8.x or above (which can be checked by running node -v)

  • Get your User ID key from your WarpJS account (sign in or sign up to create an account)

  • WarpJS is available for Linux & macOS (Windows coming soon!)

Feel free to post issues or requests regarding its prerequisites: https://github.com/WarpJS/beta/issues

If you have any trouble with node-gyp after install, please see the installation doc.
