WarpJS requirements

The WarpJS SDK has the following requirements:

  • Install Node.js latest 8.x or above (which can be checked by running node -v)

  • Create an account here. WarpJS is currently being tested by a wide community of developers. You need to get a link from a member with an active account. If you don't know any member, please request a Private Beta Access on warpjs.com by clicking on the Join Free button.

Feel free to post issues or requests regarding its prerequisites: https://github.com/WarpJS/beta/issues

If you have any trouble with node-gyp after install, please see the installation doc.
