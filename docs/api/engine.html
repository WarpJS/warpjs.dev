Warp Engine

Npm module

@warpjs/engine

https://www.npmjs.com/package/@warpjs/engine

Import

To use WarpJS stubs in the JavaScript client, you have to import engine first. The engine has to be initialize just once.

CommonJS (CJS)

require('@warpjs/engine');

ES Modules (ESM)

import '@warpjs/engine';

Script tag

Include Engine directly in an index.html file:

<script src="https://cdn.warpjs.com/warp-engine/3.0.3/engine-prod.js" type="text/javascript"></script>

Pay attention to using the same engine version that you use to develop the Node.js server part. Please change the version in the CDN url. Find current version of WarpJS thanks to the command npx warp version.
