What is WarpJS?

WarpJS is a function-as-a-service platform providing a unique backend/frontend integration feature: no backend project, you declare your backend functions in the frontend project. It runs on top of serverless providers you can select from when you put your application online: AWS lambda, GCP functions or Azure functions.

Development: WarpJS SDK lets you develop your web application in a single fullstack project with no API call nor serialization. You just warp call your functions. The warped functions are injected in the local server, allowing a live reload whether you updated the front or backend code.

Deployment: WarpJS serverless is a serverless platform you can deploy to with one command line. The WarpJS compiler splits the backend from the frontend codebase to deploy it on a serverless platform. Your frontend assets are also hosted: you don't worry about any cloud configuration.

WarpJS serverless is currently on free private beta. If you would like to test WarpJS, you need to know someone who is in the program to get an invitation link or to request an invitation on Warpjs.com.
