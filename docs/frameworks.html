Frameworks integration with WarpJS

Follow the next steps to integrate WarpJS inside the most popular frameworks: Angular, React and Vue.js.

Please note that in this tutorial we will use the CLI of each framework to generate a default web application.

Initialize your project

First, create your main project with your favorite framework:

Angular
React
Vue.js
# https://angular.io/guide/setup-local

npm install -g @angular/cli
ng new <project-name>
# https://create-react-app.dev/docs/getting-started/

npx create-react-app <project-name>
# https://cli.vuejs.org/

npm install -g @vue/cli
vue create <project-name>

Create a WarpJS module

Next, create a new module as backend. We will write a basic HTTP proxy on Node.js to fetch data from a public REST API for this example of integration.

Setup a Node.js module

Create a new my-backend directory at the root of your main project, and initialize a new npm package:

cd <project-name>
mkdir my-backend
cd server
npm init

Install the Axios library as dependency of your backend:

npm install axios

Add a function

Create a new index.js file, and init the getData() function as below:

// my-backend/index.js

const axios = require("axios");

const loadData = async function () {
  // fetch data from API
  const { data } = await axios.get("https://jsonplaceholder.typicode.com/users");
  return data;
};

// export the function from the module (to be used by another project with the "import" statement)
module.exports = { loadData };

Configure the WarpJS module

Create a new warp.config.js file at the root of your backend module:

// my-backend/warp.config.js

module.exports = {
  output: {
    format: "node-module",
    // path to the "node_modules" directory of your main project
    projectPath: "../",
    // name of the generated module
    name: "my-backend",
  },
};

Thanks to the node-module output format, WarpJS will generate an npm package named my-backend.

Why this option?

  • The node_modules is a safe directory: most frameworks use dev tools (like Babel, ESLint, …) and here, you are sure that the generated code will not be transpiled or linted.
  • Really easy to import: no relative path to know, you can directly do: import MyBackend from "my-backend"

Setup the WarpJS CLI

Go back to your main project, and install the WarpJS CLI in your project to:

  • Start the WarpJS emulator in development mode that simulate a cloud environment on localhost.
  • Build a WarpJS client in order to call the backend module.
  • Deploy the module with its exported functions and the static assets.

Install dev dependencies

cd <project-name>
npm install warp --save-dev
npm install npm-run-all --save-dev

Configure an hosting module

Create a new warp.config.js file at the root of your main project, and define the path of your public directory, in order to deploy your web app:

Angular
React
Vue.js
module.exports = {
  hosting: "angular-app",
  public: "dist/angular/",
};
module.exports = {
  hosting: "react-app",
  public: "build/",
};
module.exports = {
  hosting: "vue-app",
  public: "dist/",
};

Use the WarpJS module

Install dependencies

cd <project-name>
npm install @warpjs/engine

Call the module in your web app

Import the WarpJS engine once. We recommend to initialize it in the entry point of your main project:

Angular
React
Vue.js
// src/main.ts

import "@warpjs/engine";
// src/index.js

import "@warpjs/engine";
// src/main.js

import "@warpjs/engine";

Then import your node module, and call the exported backend function in any component:

Angular
React
Vue.js
import MyBackend from "my-backend";

const { loadData } = new MyBackend({}) as any;

loadData().then((data) => {
  console.log(data);
});
import MyBackend from "my-backend";

const { loadData } = new MyBackend();

loadData().then((data) => {
  console.log(data);
});
import MyBackend from "my-backend";

const { loadData } = new MyBackend();

loadData().then((data) => {
  console.log(data);
});

Run your project

Login to WarpJS

Run the warp login command to be authenticated with your ScaleDynamics account:

npx warp login

Create project in your account

Create a new project in your ScaleDynamics Console, or directly from the WarpJS CLI:

npx warp project create
